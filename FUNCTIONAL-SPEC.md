# AGM IR Admin — 기능명세서 (상세)

> 최종 IA 기준 모듈별 기능 정의. 화면: dashboard · shareholders · articles · settings · agenda · prevote · live · result · templates · audit
> 도메인: 주주총회 SRM 어드민 · 키컬러 블루(Park UI/Rapidpay)

---

## 1. 주주명부 관리 (shareholders)

### 1-1. 원본 업로드
- 명의개서 대리인 선택 (드롭다운)
- 주주명부 기준일 선택 (날짜)
- 원본 파일 업로드
- 업로드 직후 상태: **분류중** → 1~2 영업일 후 **확정완료**
- 상태 표시: 분류중 / 확정완료 (상태 칩)

### 1-2. 확정 후 요약 확인
확정 완료 시 다음 지표 확인:
- 총 주식수
- 총 주주수
- 운용기관 수
- 자금제공기관 수
- **미설정 전자투표권자수** → 투표권자 수동 설정 필요(액션 유도)

### 1-3. 투표권자 수동 설정
- 미설정 전자투표권자에 대해 투표권자 수동 지정

### 1-4. 통합 제외 주주 설정
- 기본: 주민등록번호 기준 통합 처리 / 투표권자 사업자번호 기준 통합 처리
- 통합이 잘못된 경우 **통합 제외** 설정:
  - 사업자번호 기준 통합이 잘못 묶인 경우
  - 우리사주조합장이 개인 내역과 합쳐진 경우
- 통합/제외 내역 검토 UI

### 1-5. 의결권 제한 처리
- 회사 법무팀 확인 내용 기준으로 다음 의결권 제한:
  - 자사주
  - 상호주
  - 무효주
  - 최대주주 및 특수관계인
- 제한 사유·근거 기록

### 1-6. 최종 확정
- 위 설정(통합 제외 + 의결권 제한)을 반영한 **최종 내용 확정 처리**
- 확정 명부만 주총 생성 시 선택 가능

## 2. 정관 관리 (articles)

### 2-1. 정관 업로드 & 결의 기준 확인
- 정관 업로드
- 주주총회 관련 내용 확인 → **결의 기준**(보통/특별결의 요건 등) 확인

### 2-2. 변경 내용 업데이트 & 이력
- 주총 진행의 **의안 규칙 설정**에 따라 정관 변경 내용 업데이트
- 수정 이력(개정 이력) 확인 — 현행 ↔ 개정안 비교

## 3. 주주총회 설정

### 3-A. 기본설정 (settings)

#### Step 1 — 기본 정보
- 기준 주주명부 선택 (확정 명부)
- 자산총액 입력/선택
- 정기 / 임시 주총 선택
- 주총 기수 입력
- 주총 일시
- 주총 장소

#### Step 2 — 진행 방식 선택
- **사전의결권** 이용 여부
  - 전자투표 이용 여부, 행사기간
  - 전자위임 이용 여부, 행사기간
- 전자위임장 이용 시:
  - 위임장 양식 등록
  - 참고서류 등록
  - 대리행사 권유자(대리인) 지정
  - 권유자 의견 공개 적용 여부
- **생중계**:
  - 이용 여부, 이용 기간
  - 질의하기 / 게시판 이용 여부
  - 공개 시점
- **현장 표결 방법**: 서면 투표 / 앱 사용
- **온라인 전자투표** 사용 여부
- ⚠️ **자산 2조원 이상 기업 → 온라인 전자투표 필수** (조건부 필수 표시)

### 3-B. 의안 설정 (agenda)
- 의안 목록 작성 (드래그 순서, 분리 의안)
- 의안 규칙 설정 (정관 연계)
- 전자위임 의향 설정 (전자위임 이용 시에만 노출)
- 의안별 의결권 제한 설정 (Stage 2 / 3%cap 등)

### 3-C. 사전의결권 관리 (prevote) — 사전의결권 이용 시
- 사전의결권 진행 내역 확인
  - 전자투표/위임 내역
  - 서면 사전투표 / 자사 서면위임장 / 타사 서면위임장 업로드
  - **중복 의결권** 확인 및 처리
  - 의결권 내역 중 **집중투표 안건 표 분배**
- 주총 전일: 사전의결권 내역 확인, 중복처리 확인/완료

---

## 4. 주주총회 진행 (live)

### 4-A. 관리자 뷰
- 사전의결권 진행 내역 데이터 확인 (정족수 확인, 의안별 집계 확인)
- 현장 참석자 관리
- 생중계 참석자 관리
- 주주총회 진행 컨트롤:
  - 주총 선언문
  - 상정문
  - 투표 시작 전 질의 접수 및 답변 질문 선정 → 의장에게 전달
  - 의안별 투표 시작 / 마감
  - 표결문
  - 주총 완료
- SSE 연결 상태 모니터링

### 4-B. 의장용 뷰
- 관리자 컨트롤에 따른 진행 상황 확인
- 참석 주주 / 주식수 확인
- 표결 진행 상황 확인
- 의안별 상정문·표결문 확인 후 진행
- 접수 질문 확인 후 답변
- 주총 진행 스크립트 확인
